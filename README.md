# hudson-nyc

A 1.6 inch scale New York Central JE-1 Hudson based on Henry Greenly's design.

## Getting Started

These instructions will get you a copy of the project on your local machine for development purposes.

### Prerequisites

What things you need

```
Fusion360
```
